Coursework

PROG 109 - Fundamentals of Web Development

Instructor: Adam Brehm

Description: Introduces fundamental programming techniques using current web development software. Students design procedures and write computer instructions to solve business problems, learn procedural programming, develop graphical user interfaces for the web and work with events and objects.

Reflection: Before this class, I knew nothing about HTML, CSS, or Javascript but after taking this course, I know how to code in those three languages because I understand the basic syntax.

MATH& 153 - Calculus III

Instructor: Sandi Xhumari

Description: Emphasizes the study of infinite sequences and series including power series. Topics include plane analytic geometry, graphing in polar coordinates, and an introduction to vectors.

Reflection: Before I took this class, I only knew about integrals and derivatives but after taking this class, I know how to use convergencse tests to determine if a series converges or diverges. I also learned how to apply vector operations.

Link to Class Catalog: MATH& 153


MATH& 254 - Calculus IV

Instructor: Michael Broome

Description: Extends the concepts of calculus to vector-valued functions and functions of several variables. Partial derivatives are included.

Reflection: Before I took this class, I was only familiar with rectangular coordinates but after taking this class, I know how to use polar and spherical coordinates. I also learned how to take partial derivatives and calculate double integrals.
